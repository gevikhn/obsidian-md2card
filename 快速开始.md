# MD2Card Obsidian 插件 - 快速开始

## 📝 项目介绍

MD2Card 是一个 Obsidian 插件，可以将你的 Markdown 笔记转换为精美的卡片图片。基于同名的 Web 应用，现在你可以直接在 Obsidian 中使用这个功能！

## ✨ 主要功能

- 🎯 **选择转换**：选中部分文本，转换为卡片
- 📄 **文档转换**：转换整个文档为卡片
- ⚙️ **自定义设置**：调整卡片宽度和高度
- 💾 **导出图片**：一键导出为 PNG 格式
- 🌐 **完整支持**：支持所有 Markdown 语法

## 🚀 快速安装

### 方法一：手动安装（推荐）

1. **下载文件**
   - 下载 `main.js`、`manifest.json`、`versions.json` 三个文件

2. **创建插件文件夹**
   ```
   你的笔记库/.obsidian/plugins/md2card/
   ```

3. **复制文件**
   - 将三个文件放入 `md2card` 文件夹

4. **启用插件**
   - 打开 Obsidian
   - 进入 设置 → 第三方插件
   - 关闭"安全模式"（如果开启）
   - 找到 "MD2Card" 并启用

### 方法二：从源码构建

```bash
# 克隆仓库
git clone https://github.com/gevikhn/obsidian-md2card.git
cd md2card

# 安装依赖
npm install

# 构建插件
npm run build:plugin

# 将生成的文件复制到 Obsidian 插件目录
```
