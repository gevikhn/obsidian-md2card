# MD2Card Obsidian 插件 - 快速开始

## 📝 项目介绍

MD2Card 是一个 Obsidian 插件，可以将你的 Markdown 笔记转换为精美的卡片图片。基于同名的 Web 应用，现在你可以直接在 Obsidian 中使用这个功能！

## ✨ 主要功能

- 🎯 **选择转换**：选中部分文本，转换为卡片
- 📄 **文档转换**：转换整个文档为卡片
- 🎨 **四种主题**：默认、暗黑、玻璃、温暖
- ⚙️ **自定义设置**：调整卡片宽度和高度
- 💾 **导出图片**：一键导出为 PNG 格式
- 🌐 **完整支持**：支持所有 Markdown 语法

## 🚀 快速安装

### 方法一：手动安装（推荐）

1. **下载文件**
   - 下载 `main.js`、`manifest.json`、`versions.json` 三个文件

2. **创建插件文件夹**
   ```
   你的笔记库/.obsidian/plugins/md2card/
   ```

3. **复制文件**
   - 将三个文件放入 `md2card` 文件夹

4. **启用插件**
   - 打开 Obsidian
   - 进入 设置 → 第三方插件
   - 关闭"安全模式"（如果开启）
   - 找到 "MD2Card" 并启用

### 方法二：从源码构建

```bash
# 克隆仓库
git clone https://github.com/gevikhn/md2card.git
cd md2card

# 安装依赖
npm install

# 构建插件
npm run build:plugin

# 将生成的文件复制到 Obsidian 插件目录
```

## 💡 使用方法

### 第一步：选择内容

在你的笔记中选择想要转换的文本：

```markdown
# 我的学习笔记

## 重要概念

- **机器学习**：让计算机从数据中学习
- **深度学习**：使用神经网络的机器学习

> "数据是新时代的石油"
```

### 第二步：执行命令

1. 按 `Ctrl+P` (Windows) 或 `Cmd+P` (Mac) 打开命令面板
2. 输入 "MD2Card"
3. 选择以下命令之一：
   - `MD2Card: Convert selection to card` - 转换选中内容
   - `MD2Card: Convert document to card` - 转换整个文档

### 第三步：预览和导出

1. 预览窗口会显示生成的卡片
2. 可以在设置中切换主题
3. 点击 "Export as Image" 导出图片
4. 图片会保存到你的下载文件夹

## 🎨 主题介绍

### 1. 默认主题
- **风格**：明亮的黄色背景，带有波点图案
- **装饰**：粉色星星和蓝色徽章
- **适用**：有趣的内容、社交媒体分享

### 2. 暗黑主题
- **风格**：紫色渐变，深色半透明叠加
- **文字**：白色高对比度
- **适用**：专业内容、技术文档

### 3. 玻璃主题
- **风格**：半透明玻璃效果
- **特点**：优雅的模糊背景
- **适用**：简约设计、现代风格

### 4. 温暖主题
- **风格**：温暖的桃色渐变
- **氛围**：舒适友好
- **适用**：个人笔记、日记

## ⚙️ 自定义设置

打开 设置 → MD2Card 可以调整：

### 卡片主题
从 4 个主题中选择：
- 默认
- 暗黑
- 玻璃
- 温暖

### 卡片宽度
- 默认：440 像素
- 建议范围：300-800 像素
- 用途：
  - 440px：适合手机
  - 500px：适合 Instagram
  - 600px：适合 Twitter

### 卡片高度
- 默认：586 像素
- 设置为 0：自动高度
- 建议范围：400-1000 像素

## 💡 使用技巧

### 最佳实践
1. **保持内容聚焦**：每张卡片只表达一个主题
2. **使用标题**：清晰的标题结构更易读
3. **善用列表**：列表在卡片格式中效果很好
4. **添加强调**：使用**粗体**和*斜体*增加视觉趣味
5. **选择合适主题**：根据内容类型选择主题

### 推荐尺寸
- **社交媒体**：440x586 或 500x500
- **演示文稿**：800x600
- **移动设备**：440x586（默认）
- **自定义**：根据需要调整

### 工作流建议
1. 写笔记时标记重要内容
2. 定期转换为卡片
3. 按主题组织导出的卡片
4. 建立卡片库用于复习

## ❓ 常见问题

**Q：能自定义主题吗？**  
A：当前版本提供 4 个内置主题，未来版本可能支持自定义主题。

**Q：支持哪些 Markdown 语法？**  
A：支持所有标准 Markdown 语法，包括表格、代码块、图片等。

**Q：可以批量生成卡片吗？**  
A：当前需要逐个转换，批量处理功能计划在未来版本中添加。

**Q：导出的图片质量如何？**  
A：使用 2x 像素比率，确保高质量输出。

**Q：能导出其他格式吗？**  
A：当前只支持 PNG 格式，未来可能添加更多格式。

## 🔧 故障排除

### 插件没有显示
- 检查文件是否在正确位置
- 重启 Obsidian
- 确认已关闭安全模式

### 命令不工作
- 确认插件已启用
- 尝试重新加载插件
- 检查是否选择了文本（对于选择转换）

### 导出失败
- 检查浏览器控制台错误
- 确认有权限写入下载文件夹
- 尝试减小卡片尺寸
